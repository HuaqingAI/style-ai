| Property | Value |

## Overview

**Page Purpose:** 林曼从相册、截图或当前页面导入一个外部参考图，并在进入生成设置前确认这就是本次要试到自己身上的对象。

**Entry Context:** 林曼已经从 `03.1-tryon-entry` 进入导入流程。她可能刚从电商、小红书或线下场景保存了截图，也可能只是想用一个示例参考先跑通试穿判断。

**Exit Action:** 确认导入参考图，进入 `03.3-tryon-setup`。

**On-Page Interactions:**
- 选择导入来源：相册/截图、当前页面分享图、示例参考图。
- 看到已导入参考图的预览、来源、类别和本次决策问题。
- 重新选择参考图或确认继续。
- 在导入失败或尚未选择时看到清晰的恢复方式。

## User State

**Hope:** 她希望不用重新解释需求，直接把眼前的心动对象放进 Style AI，尽快看到是否适合自己。

**Worry:** 她担心导入步骤太麻烦、选错图、图片质量不够，或者还没生成就已经消耗额度。

## Functional Requirements

- 页面必须明确说明：本页只导入参考图，生成额度会在下一步确认后再消耗。
- 页面必须提供至少三种导入入口：相册/截图、当前页面分享图、示例参考图。
- 页面必须展示导入前空状态、导入中状态、导入成功状态和导入失败状态。
- 导入成功后必须展示参考图预览、来源、类别、决策问题和紧迫性。
- 主行动必须在未导入时不可继续，在导入成功后进入 `03.3-tryon-setup.html`。
- 页面必须保持 Mobile-Only 单列布局，在 375px、393px、428px 宽度下无水平滚动。
