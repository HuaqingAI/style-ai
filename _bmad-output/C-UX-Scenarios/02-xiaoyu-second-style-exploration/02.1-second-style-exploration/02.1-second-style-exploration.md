# 02.1-second-style-exploration

## Page Metadata

| Property | Value |
|----------|-------|
| **Scenario** | 小雨的二次风格探索 |
| **Page Number** | 02.1 |
| **Platform** | Mobile |

## Overview

**Page Purpose:** 小雨从首次结果进入，看到可继续尝试的风格方向和本次额度消耗。

**Entry Context:** 她从首次个人形象结果页进入，看完推荐理由和谨慎点后，对结果产生了基本信任。

**Exit Action:** 选择一个新的风格方向并开始生成。

**On-Page Interactions:**
- 浏览不同的风格方向，理解每个方向和首次结果的差异。
- 查看本次继续生成会消耗的额度。
- 选择一个更日常或更明显不同的风格方向。

## First-Step Context

**User Situation:** 小雨刚看完第一次个人形象结果，觉得其中一个方向“有点像自己，也能判断”，但还不确定是否最适合。

**Mental State:**
- **Hope:** 她希望趁第一次结果还有参考价值时，再看一个不同方向，比较哪种更适合自己。
- **Worry:** 她担心继续生成会浪费额度，或者结果之间没有明显差异，无法帮助她判断。

**Discovery Method:** 她看到“换一个风格看看”或类似继续探索入口，于是决定消耗一次额度再生成一个方向。
