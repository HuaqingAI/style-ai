# Product Brief 对话上下文

## 项目元信息

- 项目名称：style-ai
- 当前状态：现有代码仅作为 demo 参考，Product Brief 按从 0 到 1 的新产品分析处理
- 沟通语言：中文
- 文档输出语言：中文
- 当前阶段：WDS Phase 1 Product Brief

## 已有项目背景

README 中的 demo 描述为一款个人形象 AI 助理：用户通过拍照或上传照片，让多模态 AI 分析发型、穿搭、妆容、配饰，并获得评分、改进建议、历史记录与个人偏好化反馈。

## 已确认方向

- 首批用户：日常变美的新手
- 暂不做专用场景或专业造型师工具
- 初始核心场景：
  - 此时此刻基于当前形象，给出未来改变建议
  - 买衣服试穿时，帮助判断是否适合、是否值得买
