# 04.1-result-quick-feedback-panel

## Page Metadata

| Property | Value |
|----------|-------|
| **Scenario** | 小雨的种子用户反馈 |
| **Page Number** | 04.1 |
| **Platform** | Mobile |

## Overview

**Page Purpose:** 小雨在结果页内选择“像不像、能不能判断、哪里不对”等轻量反馈项。

**Entry Context:** 她从个人形象结果页或试穿结果页看到轻量反馈入口，在继续探索前顺手点开。

**Exit Action:** 提交反馈后回到继续探索或重试。

**On-Page Interactions:**
- 选择“像不像本人”。
- 选择“能不能用于判断”。
- 根据当前结果类型动态出现更具体的快捷项：如“脸不像本人”“发型不自然”“不像参考图”“比例不对”“照片角度影响结果”。
- 标记具体问题：不像本人、风格跑偏、照片问题、无法判断或其他。
- 提交反馈并返回原结果上下文。

## First-Step Context

**User Situation:** 小雨刚看完一次生成结果，觉得结果整体有价值，但其中某些地方不像自己、风格略跑偏，或者她不确定能不能真的拿来判断。

**Mental State:**
- **Hope:** 她希望轻松告诉产品哪里像、哪里不像，让下一次生成更准。
- **Worry:** 她担心反馈太麻烦，或者说了也不会影响结果。

**Discovery Method:** 入口不是强制弹窗，而是贴着结果判断出现，让她可以快速表达“像不像、能不能判断、哪里不对”。
