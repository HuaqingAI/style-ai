# 01.1-start-login

## Page Metadata

| Property | Value |
|----------|-------|
| **Scenario** | 小雨的首次真实预览 |
| **Page Number** | 01.1 |
| **Platform** | Mobile |

## Overview

**Page Purpose:** 小雨用种子测试账号进入，并理解登录和额度、历史、账号管理相关。

**Entry Context:** 小雨晚上刷小红书时又收藏了几个风格参考，想到自己一直判断不出来，就从 Sue 发来的种子测试邀请打开 Style AI 手机 App。

**Exit Action:** 登录成功后进入首页。

**On-Page Interactions:**
- 查看登录必要性的简短说明：登录用于额度、历史和账号管理，不等于强制建立长期个人形象档案。
- 使用手机号、验证码或种子测试账号完成登录。

## First-Step Context

**User Situation:** 小雨，26 岁，一线或新一线城市白领。她刚刷到几个心动的发型/穿搭参考，但越看越不确定自己适不适合。

**Mental State:**
- **Hope:** 她希望先看到一个足够像自己的风格预览，判断这个方向现实里能不能尝试。
- **Worry:** 她担心照片被默认长期保存，或者生成结果不像本人反而误导判断。

**Discovery Method:** 她从 Sue 发来的种子测试邀请打开 Style AI 手机 App，不是带着购买任务进来，而是想先验证“这个工具能不能让我看见自己”。
