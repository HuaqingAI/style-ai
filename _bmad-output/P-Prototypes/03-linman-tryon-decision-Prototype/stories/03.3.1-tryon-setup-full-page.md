# 03.3.1 Tryon Setup Full Page

## Purpose

Build `03.3-tryon-setup.html` as a mobile prototype draft for confirming reference image, profile image, intent, and credits before generation.

- `_bmad-output/C-UX-Scenarios/03-linman-tryon-decision/03-linman-tryon-decision.md`

## Objects

- `obj-setup-header`: top navigation with back path to `03.2-import-reference.html`.
- `obj-setup-hero`: concise explanation of final pre-generation confirmation.
- `obj-setup-reference-summary`: imported reference summary and decision question.
- `obj-setup-profile-card`: current profile image state and alternate missing profile state.
- `obj-setup-intent-control`: clothing / hairstyle / makeup segmented control.
- `obj-setup-credit-confirmation`: balance, cost, and failure expectation copy.
- `obj-start-generation-action`: primary CTA, enabled only in default / submit-error retry states.
- `obj-setup-state-controls`: local reviewer controls for page states.

## State Behavior

- `default`: profile ready, balance enough, start action enabled.
- `missing-profile`: profile unavailable, start action disabled.
- `low-credit`: balance below required cost, start action disabled.
- `submitting`: start action disabled and shows in-progress copy.
- `submit-error`: start action enabled as retry and shows recovery copy.

## Layout Requirements

- Mobile-only single column, max width `428px`.
- Use dark Style AI surface `#111111`, ivory text `#FAFAF7`, spark accents `#7B52FA` and `#9D7AFF`.
- Cards use `rounded-lg`, light borders, compact mobile text scale.
- No horizontal scroll at 375px.

## JavaScript Requirements

- Implement `setSetupState(state)`.
- Implement intent switching with visible active state.
- Disable forward navigation when state is `missing-profile`, `low-credit`, or `submitting`.
- In default state, primary action links to `03.4-tryon-waiting.html`.
- Expose `window.__STYLE_AI_PROTOTYPE__` with state setter for automated verification.

## Acceptance Criteria

Agent-verifiable:
- All object IDs exist.
- Five states can be triggered.
- Intent control has three options and active state updates.
- Default state enables `obj-start-generation-action` and points to `03.4-tryon-waiting.html`.
- Missing-profile / low-credit / submitting states disable forward navigation.
- No console errors at load and during state switching.
- No horizontal overflow at 375px, 393px, and 428px.

User-evaluable:
- It is clear that this is the last checkpoint before generation.
- It is clear what reference object and personal image will be used.
- Credit cost is visible without making the page feel like a purchase page.
